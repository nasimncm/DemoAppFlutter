# Images Folder

Yahan aap apni image files add karein.

## Image Files Required:

### Profile Images:
- `zeeshan.png` (or .jpg)
- `app-icon.png` (or .jpg)

### Post Images:
- `party-video.png` (or .jpg)
- `last-image.png` (or .jpg)
- `group.png` (or .jpg)
- `party.png` (or .jpg)
- `party-2.png` (or .jpg)
- `group2.png` (or .jpg)

## Image Names Used in Code:

**Profile Images:**
- `zeeshan` → `assets/images/zeeshan.png`
- `app-icon` → `assets/images/app-icon.png`

**Post Images:**
- `party-video` → `assets/images/party-video.png`
- `last-image` → `assets/images/last-image.png`
- `group` → `assets/images/group.png`
- `party` → `assets/images/party.png`
- `party-2` → `assets/images/party-2.png`
- `group2` → `assets/images/group2.png`

## Supported Formats:
- `.png` (recommended)
- `.jpg` / `.jpeg`

## Note:
- Image file names case-sensitive hote hain
- Agar image file nahi milegi, to placeholder icon dikhega
- Images ko `assets/images/` folder mein directly rakhna hai (subfolder mein nahi)

## Example Structure:
```
assets/
  images/
    ├── zeeshan.png
    ├── app-icon.png
    ├── party-video.png
    ├── last-image.png
    ├── group.png
    ├── party.png
    ├── party-2.png
    └── group2.png
```
